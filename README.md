# vgm_converter

The library can be used to convert VGM / NSF data to PCM on the fly.

It recognizes:
 * vgm files (AY-3-8910, YM2149 (MSX2) and NES APU (Nes console))
 * nsf files (Nintendo Sound Format - NES APU)

# Compilation

> make

> ./vgm2pcm
